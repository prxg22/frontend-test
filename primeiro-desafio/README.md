# Primerio Desafio
Create HTML/CSS page

## Quickstart

1. Install `npm` dependecies

    `$ npm install`

2. Move to `src/` folder

    `$ cd src`

3. Install `bower` dependecies

    `$ bower install`

4. Start server to development

    `$ gulp`


## Deploy

1. Build project on `dist/` folder

    `$ gulp build`

2. Build and serve on `dist/` folder
    `$ gulp server:build`
