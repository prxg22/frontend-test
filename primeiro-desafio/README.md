# boilerplate
Angular, SASS and Gulp front-end projects boilerplate

## Quickstart

- Clone the repository

- Install `npm` dependecies

    `$ npm install`

- Move to `src/` folder

    `$ cd src`

- Install `bower` dependecies

    `$ bower install`

- Start server to development

    `$ gulp`

- Build project on 'dist/' folder

    `$ gulp build`
